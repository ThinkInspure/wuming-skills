## 项目概述

这是一个帮助用户深入理解 GitHub 开源项目的智能助手 skill。它能够：

1. 自动分析 GitHub 仓库的结构和代码
2. 生成两份高质量文档：
   - 源码与原理解读报告（深度分析）
   - 快速上手文档（实用指南）
3. 自动安排 1 小时后的复查任务
4. 生成 Mermaid 图表可视化架构

## 文件结构

```
github-code-interpreter/

```

## 已知限制

### 1. URL 格式

目前只支持标准 HTTPS URL：
- ✅ `https://github.com/user/repo`
- ✅ `https://github.com/user/repo.git`
- ❌ `git@github.com:user/repo.git` (SSH)
- ❌ 带子路径的 URL

### 2. 仓库大小

超大项目可能需要较长时间：
- 建议 > 100K 行代码的项目聚焦某个模块

### 3. 依赖工具

需要系统安装 `git` 和 `tree`（可选）
