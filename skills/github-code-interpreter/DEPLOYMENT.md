## 📝 注意事项

### 使用限制

1. **URL 格式**: 只支持标准 HTTPS URL
2. **项目大小**: 超大项目需要聚焦模块
3. **依赖工具**: 需要 git 和 tree（可选）

### 已知问题

1. 正则表达式可能无法解析某些特殊 URL
2. 大仓库统计可能较慢
3. 某些系统可能没有 tree 命令

### 改进建议

1. 使用专门的 URL 解析库
2. 添加超时机制
3. 提供更多统计选项
